# DashRAG
Repo for MVP2 of Team DASH's ECE496 Capstone Project.

## Quick-Start
1. Clone the repo.
2. Open two terminals, one for the back-end and one for the front-end.
3. First, for the back-end terminal:
    - Change directory to `\src\api\`
    - Follow the instructions in the `README.md` file in this directory
4. For the front-end terminal:
    - Change directory to `\src\front-end\
    - Do `npm install`
    - Run `npm run dev` to launch the front-end.
